Redhat更换yum源

Linux
Linux
发布日期:   2018-08-14
文章字数:   621
阅读次数:  

简述

redhat没有进行注册（收费了），不能够使用其自带的yum源，所以需要第三方的源来代替。

阿里云镜像源

查看自带yum源

rpm -qa | grep yum
# 查询结果
yum-utils-1.1.31-24.el7.noarch
yum-langpacks-0.4.2-3.el7.noarch
yum-metadata-parser-1.1.4-10.el7.x86_64
yum-rhn-plugin-2.0.1-4.el7.noarch
PackageKit-yum-0.8.9-11.el7.x86_64
yum-3.4.3-118.el7.noarch

(这些安装包都需要从阿里源重新下载)

卸载RedHat自带yum源

rpm -qa|grep yum|xargs rpm -e --nodeps

配置redhat DNS解析（否则不能解析域名）

vim /etc/sysconfig/network-scripts/ifcfg-lo

#ifcfg-lo文件添加DNS映射
DNS1=8.8.8.8
DNS2=8.8.8.4

使用wget将需要的安装包从阿里yum源
下载下来

wget https://mirrors.aliyun.com/centos/7/os/x86_64/Packages/yum-3.4.3-158.el7.centos.noarch.rpm
...
...

安装

rpm -ivh yum-3.4.3-158.el7.centos.noarch.rpm --nodes --force

建议一个一个安装

新建配置文件(用于阿里源配置)
vim /etc/yum.repos.d/CentOS-Base.repo

baseurl=https://mirrors.aliyun.com/centos/7/os/$basearch/
gpgcheck=1
gpgkey=https://mirrors.aliyun.com/centos/RPM-GPG-KEY-CentOS-7

#released updates
[updates]
name=CentOS-$7 - Updates - aliyun.com
#mirrorlist=http://mirrorlist.centos.org/?release=$7&arch=$basearch&repo=updates
baseurl=https://mirrors.aliyun.com/centos/7/updates/$basearch/
gpgcheck=1
gpgkey=https://mirrors.aliyun.com/centos/RPM-GPG-KEY-CentOS-7

#additional packages that may be useful
[extras]
name=CentOS-$7 - Extras - aliyun.com
#mirrorlist=http://mirrorlist.centos.org/?release=$7&arch=$basearch&repo=extras
baseurl=https://mirrors.aliyun.com/centos/7/extras/$basearch/
gpgcheck=1
gpgkey=https://mirrors.aliyun.com/centos/RPM-GPG-KEY-CentOS-7

#additional packages that extend functionality of existing packages
[centosplus]
name=CentOS-$7 - Plus - aliyun.com
baseurl=https://mirrors.aliyun.com/centos/7/centosplus/$basearch/
gpgcheck=1
enabled=0

gpgkey=https://mirrors.aliyun.com/centos/RPM-GPG-KEY-CentOS-7

下载软件测试下

yum -y install tree

本地iso镜像源

上传文件到服务器目录rhel-server-6.5-x86_64-dvd-1.iso

# 创建文件
mkdir /mnt/yum
# 挂载镜像
mount -o loop rhel-server-6.5-x86_64-dvd-1.iso /mnt/yum

修改配置文件vim /etc/yum.repos.d/rhel.repo

[redhat]
# 改为1
enabled=1
# 文件挂载位置
baseurl=file:///mnt/yum
#其他配置不用改

测试

yum clean
yum install tree

用过的命令

RPM—>Redhat package Manager软件管理工具

常用命令：

  • -ivh : 安装显示安装进度 –install–verbose–hash
  • -Uvh : 升级软件报 –Update
  • -e : 删除包
  • -qpl ：列出RPM软件包内的文件信息Query Package list
  • -qpi ：列出RPM软件包的描述信息Query Package install package(s)
  • -qf ：查找指定文件属于哪个RPM软件包Query File
  • -Va：校验所有的RPM软件包，查找丢失的文件View Lost
